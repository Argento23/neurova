## 1. OBJETO DEL CONTRATO
El presente acuerdo establece los términos bajo los cuales **GenerArise** implementará, desplegará y mantendrá soluciones de **Automatización basada en Inteligencia Artificial** (en adelante, "Los Agentes") para **El Cliente**.

El objetivo es la optimización de procesos comerciales, cualificación de leads y/o gestión de datos mediante el uso de Modelos de Lenguaje Grande (LLMs) y flujos programáticos.

## 2. INFRAESTRUCTURA Y SOPORTE TECNOLÓGICO (GOOGLE CLOUD COMPLIANCE)
GenerArise garantiza que la infraestructura subyacente de "Los Agentes" corre sobre estándares de nivel empresarial (Enterprise Grade).

### 2.1 Stack Tecnológico
Para garantizar la estabilidad, escalabilidad y seguridad, utilizamos proveedores de clase mundial:
*   **Motor de Inteligencia:** Google Cloud Vertex AI / OpenAI API (Modelos GPT-4 / Gemini 1.5 Pro).
*   **Infraestructura de Servidores:** Google Cloud Run (Serverless architecture) para garantizar un uptime del 99.9%.
*   **Orquestación:** n8n (Workflow Automation) alojado en servidores privados virtuales (VPS) con encriptación SSL.

### 2.2 Seguridad de Datos
*   Los datos procesados por nuestros agentes **NO** se utilizan para entrenar modelos públicos.
*   Cumplimiento con estándares de encriptación en tránsito (TLS 1.2+) y en reposo (AES-256).

## 3. ALCANCE DEL SERVICIO (SLA)

### 3.1 Responsabilidades de GenerArise
*   Diseño y programación de los flujos de automatización.
*   Integración con el CRM o plataformas del cliente (WhatsApp, Email, etc.).
*   Mantenimiento correctivo en caso de fallos de API o desconexiones.
*   **Tiempo de respuesta de soporte:** Máximo 24 horas hábiles para incidencias críticas.

### 3.2 Limitación de Responsabilidad (Alucinaciones de IA)
El Cliente reconoce que la tecnología de Inteligencia Artificial Generativa es probabilística. Aunque GenerArise implementa filtros de seguridad ("Guardrails"), existe una posibilidad técnica no nula de que el Agente genere respuestas inexactas ("alucinaciones").
*   GenerArise no se hace responsable por pérdidas comerciales derivadas de una interpretación errónea de un texto generado por la IA.
*   Se recomienda siempre supervisión humana en las primeras fases del despliegue.
